# App Builder Overview

## Introduction

App Builder is a comprehensive framework enabling enterprise developers to create and deploy custom web applications that extend Adobe Experience Cloud solutions. These applications operate on Adobe infrastructure, utilizing Adobe I/O Runtime—a serverless, event-driven computing platform.

## Target Audience

App Builder serves Adobe enterprise customers and their partner organizations seeking to enhance Adobe Experience Cloud capabilities.

## Documentation Structure

The documentation provides guidance organized by user type:

### For Business Users and Decision-Makers

Resources include:
- Business case outlining agility, efficiency, and security benefits
- Real-world implementation examples from Adobe customers and partners
- FAQ section addressing licensing and trial information

### For Developers

The framework supports learning across multiple phases:

**Exploration Phase**
- Design concepts and foundational principles
- Core architectural components
- Integration with the development lifecycle

**Learning Phase**
- Hands-on tutorials for building custom applications
- Adobe I/O Runtime setup and deployment guidance
- Performance optimization strategies

**Problem-Resolution Phase**
- Comprehensive FAQ resources
- Topic-organized technical guides with reference documentation
- Video tutorials and step-by-step code labs
- Community contribution opportunities through developer forums and open-source repositories

## Getting Started

New developers can begin by reviewing foundational materials or diving directly into practical application building through the "Get Started with App Builder" tutorial.
